About

Welcome to my portfolio. This project is currently a work in progress, with a strong focus on building a solid backend infrastructure and deploying a functional web application in a real cloud environment.

The goal of this project is not just presentation, but to demonstrate practical experience with deployment, system design, and full-stack development.

Technologies Used

  • AWS EC2 – Provisioning and managing cloud-based virtual servers. Configured with a custom domain using DNS.
  • Python – Core programming language used for backend logic.
  • Django – Web framework used to build and structure the application.
  • Gunicorn – WSGI HTTP server used to run the Django application in production.
  • Nginx – Reverse proxy server used for handling requests, routing traffic, and improving performance.
